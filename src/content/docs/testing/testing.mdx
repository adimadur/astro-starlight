## Steps

<Steps>

1. **Install dependencies**
   
   Run the following command to install Starlight:
   
   ```bash
   npm install @astrojs/starlight
   ```

2. **Configure your theme**
   
   Create a `custom.css` file with your lime theme variables:
   
   ```css
   :root {
     --sl-color-accent: #84cc16;
   }
   ```

3. **Add custom CSS**
   
   Update your `astro.config.mjs`:
   
   ```js
   export default defineConfig({
     integrations: [
       starlight({
         customCss: ['./src/styles/custom.css']
       })
     ]
   });
   ```

4. **Test your theme**
   
   Run the dev server and check your site:
   
   ```bash
   npm run dev
   ```

</Steps>
